# Instruksi Menambahkan Poster

1. Simpan file poster dengan nama: `poster.jpg`
2. Letakkan file di folder `public/`
3. Format yang didukung: JPG, PNG, atau WebP
4. Ukuran disarankan: minimal 800px lebar untuk kualitas baik di mobile dan desktop

**Catatan:** Jika file poster tidak ditemukan, gambar akan disembunyikan secara otomatis.
